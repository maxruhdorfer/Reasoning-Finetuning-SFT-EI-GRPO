# Reasoning-Finetuning-SFT-EI-GRPO
Systematic study of different reasoning finetuning methods: supervised finetuning (SFT), expert iteration (EI) and RLVR with the GRPO algorithm. Algorithms are implemented from scratch, inspired by homework assignments of Stanford's CS336 class, and are used to finetune Qwen2.5-Math-1.5B on the MATH dataset.
